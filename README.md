# RIDTT

Role-Inverted Dual-Tree Topology (RIDTT)

A preliminary research project exploring orthogonal dual-tree communication structures for fault-tolerant distributed aggregation and distributed machine learning systems.

## Core Idea

The topology constructs two complementary spanning trees over the same logical node set while minimizing persistent node criticality across both trees.

The design introduces:
- dual complementary tree layers,
- anti-correlated node criticality,
- explicit inter-tree identity connections,
- alternate routing paths for resilience.
